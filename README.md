# Портфолио аналитика данных

## Обо мне

Я аналитик данных. В портфолио собраны проекты, выполненные в рамках обучения и самостоятельной практики.

В проектах провожу исследовательский анализ данных (EDA), проверяю статистические гипотезы, анализирую пользовательское поведение и формулирую выводы и рекомендации для бизнеса.
