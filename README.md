# Портфолио аналитика данных

## Обо мне

Я начинающий аналитик данных. В портфолио собраны проекты, выполненные в рамках обучения и самостоятельной практики.

В проектах провожу исследовательский анализ данных (EDA), проверяю статистические гипотезы, анализирую пользовательское поведение и формулирую выводы и рекомендации для бизнеса.

---

## Проекты

| № | Проект | Описание | Стек |
|---|--------|----------|------|
| 1 | [Анализ результатов A/B-теста](./ab-test-results-analysis) | Расчёт параметров эксперимента, проверка корректности проведения и анализ результатов A/B-теста. Новый алгоритм показал статистически значимый рост метрики | pandas, numpy, scipy, statsmodels, matplotlib, seaborn |
| 2 | [Анализ инвестиций в стартапы](./startup-investment-analysis) | Анализ структуры и динамики венчурного финансирования стартапов, выявление перспективных сегментов и типов инвестиций | pandas, matplotlib, seaborn, phik |
| 3 | [Анализ пользовательского спроса (Яндекс Афиша)](./yandex-afisha-user-behavior-analysis) | Анализ сезонности, популярных категорий и поведения пользователей (mobile/desktop), проверка гипотез | pandas, numpy, scipy, seaborn |
| 4 | [Анализ сервиса Яндекс Книги](./yandex_books_analysis) | Проверка гипотезы о различиях во времени активности пользователей Москвы и Санкт-Петербурга. Статистически значимых различий не выявлено | pandas, numpy, scipy, matplotlib, seaborn |
