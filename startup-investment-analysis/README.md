# Исследование стартапов

## Описание проекта

Финансовая компания, работающая с венчурными инвестициями, стремится лучше понять закономерности финансирования стартапов и оценить перспективы выхода на рынок с покупкой и развитием компаний. 

Задача — провести исследование на исторических данных: изучить динамику и структуру финансирования стартапов для оценки инвестиционных стратегий.

## Навыки и инструменты

- **pandas** — обработка и анализ данных
- **matplotlib / seaborn** — визуализация
- **phik** — расчет корреляций

## Общий вывод

Анализ данных до 2015 года показал устойчивый рост венчурного рынка с 2005 года, подтвердил ключевую роль технологического сектора и высокую эффективность венчурного финансирования. 

Итоговые рекомендации предлагают сфокусироваться на массовых сегментах (technology, saas, internet) и типах финансирования с максимальной отдачей (venture, private equity).
