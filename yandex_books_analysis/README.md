# Анализ гипотезы о времени активности пользователей в Москве и Санкт-Петербурге в Яндекс Книги

## Описание проекта

Требуется принять статистически обоснованное решение о том, проводят ли пользователи из Санкт-Петербурга больше времени за чтением и прослушиванием книг в приложении Яндекс Книги, чем пользователи из Москвы.

## Используемые библиотеки

- pandas
- numpy
- scipy.stats
- matplotlib
- seaborn

## Общий вывод

Нет достаточных статистических оснований утверждать, что пользователи из Санкт-Петербурга проводят в приложении больше времени, чем пользователи из Москвы
