# Анализ гипотезы о времени активности пользователей в Москве и Санкт-Петербурге в Яндекс Книге

**Автор:** Валеева Евгения Маратовна  
**Дата:** 11.02.2026

## Описание проекта

В данном исследовании используется набор данных yandex_knigi_data.csv, который содержит информацию о пользователях из Москвы и Санкт-Петербурга с суммой часов их активности за период с 1 сентября по 11 декабря 2024 года.

Цель проекта: провести статистический анализ односторонней гипотезы с двумя выборками и принять статистически обоснованное решение о том, проводят ли пользователи из Санкт-Петербурга больше времени за чтением и прослушиванием книг в приложении Яндекс Книги, чем пользователи из Москвы.

## Описание данных

Файл yandex_knigi_data.csv содержит следующие столбцы:
- city — город пользователя (Москва / Санкт-Петербург)
- puid — идентификатор пользователя
- hours — общее количество часов активности

## Основные результаты

- После удаления дубликатов в данных осталось 8540 уникальных пользователей (6234 из Москвы, 2306 из Санкт-Петербурга)
- Доля неактивных пользователей (0–10 минут за период) составила 32.6% в обоих городах
- p-value теста Уэлча = 0.27 (> 0.05)
- Вывод: нет достаточных статистических оснований утверждать, что пользователи из Санкт-Петербурга проводят в приложении больше времени, чем пользователи из Москвы
