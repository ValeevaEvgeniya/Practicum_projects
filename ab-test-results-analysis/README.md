# Разработка A/B-тестирования и анализ результатов

## Описание проекта

Заказчик — развлекательное приложение с функцией «бесконечной» ленты.
Разработчики создали новый алгоритм рекомендаций, который, по их мнению, будет показывать пользователям более интересный контент.

## Задачи проекта:

- Рассчитать параметры A/B-теста (длительность и размер выборки).
- Оценить корректность проведения эксперимента.
- Проанализировать результаты и сделать вывод об эффективности нового алгоритма.

## Навыки и инструменты

- pandas
- numpy
- scipy.stats
- statsmodels.stats.proportion
- statsmodels.stats.power
- statsmodels.stats.proportion
- from math
- matplotlib
- seaborn

## Общий вывод

Мы выявили, что количество успешных сессий в тестовой выборке примерно на 1.1 п.п. выше, чем в контрольной (30.8% против 31.8%).  
Проведенный статистический тест показал, что p-value = 0.00016 , что значительно меньше уровня значимости 0.05.  
Это означает, что мы доказали альтернативную гипотезу и новый алгоритм рекомендаций действительно показывает более интересный контент для каждого пользователя.
